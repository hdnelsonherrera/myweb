Pasos para agregar Firebase a un proyecto de React JS

1.- Ingresar a la consola de Firebase
2.- Creamos un nuevo proyecto o tomamos alguno ya creado.
    2.1.- Opcionalmente podemos habilitar google Analythics
3.- Añadimos firebase a nuestra aplicación iOS, Android o Web
4.- Registramos la aplicación correspondiente, en este caso web, nos mostrara un JSON de configuración el cual necesitamos copiar.
5.- Guardamos la configuración en un archivo JS dentro del proyecto y la exportamos por defecto. (firebaseConfig.js)
6.- Dentro del proyecto ejecutamos el siguiente comando, para instalar las dependencias correspondientes.
    npm install --save firebase reactfire@next
7.- Importamos FirebaseAppProvider, en el archivo principal (index.js), para envolver toda la aplicación. (import { FirebaseAppProvider } from 'reactfire';)
8.- Envolvemos toda la app en el elemento del paso aterior y como prop le pasamos la configuración del paso 5.
